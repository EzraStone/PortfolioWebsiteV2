# Tower of Hanoi State Space Visualizer

This project is an interactive web application designed to visualize the state space and optimal solution paths of the Tower of Hanoi puzzle. It helps users understand the recursive structure of the puzzle by displaying possible moves, state transitions, and step-by-step solutions in a dynamic and animated interface.

Key features:
- Visual representation of each disk move and tower state
- Exploration of all valid moves within the state space
- Animated solution walkthroughs
- Interactive controls for adjusting puzzle complexity
- Educational insights into recursion and algorithmic problem solving
