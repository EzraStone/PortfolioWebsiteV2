**Chaos Equations Visualizer** is an interactive web application designed to explore and visualize chaotic systems through real-time simulations. This project provides an educational platform for understanding the fascinating and unpredictable nature of chaotic systems in mathematics and physics.

## Project Overview

The Chaos Equations Visualizer allows users to experiment with various chaotic systems including the Lorenz System, Halvorsen System, Thomas System, and many others. Users can interactively adjust parameters such as initial conditions, time steps, and resolution to observe how these changes affect system behavior in real-time.

## Key Features

### Interactive Controls
- **System Selection**: Choose from multiple chaotic systems including Lorenz, Halvorsen, Thomas, and more
- **Parameter Adjustment**: Modify initial conditions, time steps, and resolution
- **Real-time Visualization**: Dynamic plots that update as parameters change
- **3D Navigation**: Mouse controls for rotating, zooming, and panning the view

### Supported Systems
- Lorenz System
- Halvorsen System  
- Thomas System
- Chen-Lee Attractor
- Aizawa Attractor
- Rössler System
- And many more chaotic systems

### User Interface
- **Intuitive Controls**: User-friendly interface accessible to both beginners and advanced users
- **Interactive 3D View**: Click and drag to rotate, scroll to zoom, right-click to pan
- **System Selector**: Dropdown menu to switch between different chaotic systems
- **Real-time Updates**: Immediate visual feedback when adjusting parameters
